3️⃣ Fetching Data

When we fetch the data we use the Fetch API from the package 'isomorphic-fetch', this is wrapped in an async function to allow to use Promise.all on all the requests.

This allows us to 'pause' the build while we wait for the data to be fetched. ⏬ 4/9
