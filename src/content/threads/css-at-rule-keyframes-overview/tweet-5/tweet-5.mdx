4️⃣ Valid Keyframes

If a keyframe ruleset doesn't contain a start or end deceleration (from/to or 0%/100%) then the browser takes the styles for the start/end.

This means if we had a blue square and set 50% to yellow, it goes from blue at 0 to yellow at 50 & blue at 100.
