▶️ Promise.then(), .catch() &amp; .finally()

These 3 methods make up the bulk of handling a promise.

1️⃣ .then's can chain promises, passing one value to another.
2️⃣ .catch is used to handle any rejects anywhere in the chain.
3️⃣ .finally runs regardless of the outcome. ⏬ 8/10
